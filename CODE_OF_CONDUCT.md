# Code of Conduct

This project — and every repository owned by [@FlexNetOS](https://github.com/FlexNetOS)
that inherits this file — adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The canonical text, including the full enforcement ladder and the list of
behaviours considered unacceptable, is maintained at the link above.

## Reporting

Report concerns privately to **<revenaugh.david@gmail.com>** with subject
`CONDUCT: <short summary>`.

Reports are acknowledged within **48 hours**, investigated confidentially,
and resolved by the project maintainer. Reporters are kept informed of the
outcome and are not subject to retaliation for good-faith reports.

## Scope

This Code applies to all project spaces — repositories, issue trackers,
pull requests, Discussions, and any official communication channel — and
when an individual is representing the project in a public space.
